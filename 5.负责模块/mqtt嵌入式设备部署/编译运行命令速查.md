# 色选机工程 编译运行命令速查

这个是对shell工程使用的

## 环境初始化（每次开机一次）


```bash
~/桌面/aiuse-qtrun/setup_env.sh
```

## 编译

**全量编译（改了 3rdparty 或 uikits 时）：**
```bash
cd ~/桌面/aiuse && qmake solution.pro && make -j$(nproc)
```

**单独编译主工程（只改了 zksort/ 下代码）：**
```bash
cd ~/桌面/aiuse/zksort && qmake zksort.pro && make -j$(nproc)
```

## 运行

```bash
cd ~/桌面/aiuse/zksort/dist && LD_LIBRARY_PATH=../../3rdparty/log4qt/lib/unix:../../3rdparty/qextserialport/lib/unix:../../3rdparty/jsoncpp/lib/unix:../../uikits/lib/unix ./zksort
```
